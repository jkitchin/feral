## Next Session Should

1. **Decide Phase C scope.** The bench gate is the only criterion still
   open. Pick one of:
   - **ABI-break:** make `FrontalFactors` borrow scratch (touches every
     caller — expensive and may cascade).
   - **`take_into_factors(&mut self, ...)`:** swap pooled buffers into
     an owned-output struct (intermediate: caller-visible but no ABI
     break on existing fields).
   - **`with_capacity` hints:** preallocate `FrontalFactors` fields from
     an upper-bound estimate provided by symbolic analysis (cheapest;
     does not eliminate allocations, just amortises them).
