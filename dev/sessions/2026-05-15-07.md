## Decisions Made

- **PyO3 + maturin over ctypes/cffi**. PyO3 carries the
  numpy / scipy interop, structured exceptions, and Inertia
  pyclass for ~10× less binding code than a hand-written ctypes
  wrapper would. Build via maturin (PEP 517 backend) so wheel
  metadata, abi3 tags, and sdist all come for free. Decision
  body in `dev/plans/python-interface.md`.

- **abi3-py310 single-wheel-per-platform**. One wheel covers
  3.10+ on each (os, arch). Halves the cibuildwheel matrix
  vs per-minor wheels and shields us from cp3.14 ABI churn.

- **KktSolver requires explicit zero diagonals in the CSC
  pattern**. Caller must populate (2,2)-block zero diagonals
  in `from_triplet` so the perturbation loop has slots for δ_c.
  The Python-side helpers (`build_kkt_pattern`,
  `_kkt_with_explicit_zero_diag`) demonstrate the idiom; the
  HS071 example mirrors it. Alternative — silently inserting
  the slots from inside KktSolver — was rejected because it
  would mask pattern bugs from upstream pattern builders.

- **Trusted PyPI publishing only**, no API tokens. The workflow
  uses `pypa/gh-action-pypi-publish@release/v1` with
  `id-token: write` so PyPI accepts via OIDC. No long-lived
  secrets to rotate.

## Abandoned Approaches

- **`as_slice()` on numpy inputs** — required contiguous
  storage and failed on `X[:, j]` column slices in
  `test_multi_rhs_solve`. Replaced with copy via
  `as_array().iter().copied().collect::<Vec<f64>>()` for every
  user-facing array argument. Copy cost is negligible relative
  to factor; assuming contiguity is the wrong default for a
  Python API.

- **`feral::error::FeralError` re-exposed under the same name**
  via `create_exception!` — orphan-rule collision. Fix:
  aliased Rust enum as `RustFeralError`, used `FeralError` as
  the PyO3 exception name. Documented in the journal.

- **Building `python/` as a standalone `cargo build`** — PyO3
  extension-module skips libpython linkage, so cargo can't link.
  Use `maturin build` end-to-end; cargo invocation only useful
  for `cargo check`.
